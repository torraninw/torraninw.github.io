# เปลี่ยนมิเตอร์เป็นแบบ TOU คุ้มไหม? (TOU vs Normal Meter Break-Even Calculator)

แอปพลิเคชันหน้าเว็บสำหรับช่วยตัดสินใจว่า **ควรเปลี่ยนมิเตอร์ไฟฟ้าจากแบบปกติ (อัตราก้าวหน้า) ไปเป็นแบบ TOU (Time-of-Use) หรือไม่** การเปลี่ยนมิเตอร์มีค่าธรรมเนียมครั้งเดียว แต่จะคุ้มหรือไม่ขึ้นอยู่กับว่าผู้ใช้ย้ายการใช้ไฟไปช่วง Off-Peak (กลางคืน 22:00–09:00 และวันหยุด) ได้มากพอหรือเปล่า เครื่องมือนี้เปรียบเทียบค่าไฟทั้งปีของทั้งสองมิเตอร์จากพฤติกรรมการใช้ไฟชุดเดียวกัน แล้วบอกว่าคุ้มและคืนค่าเปลี่ยนมิเตอร์ภายในกี่ปี — และจะบอกตรง ๆ ว่า **ไม่ควรเปลี่ยน** ถ้าใช้ไฟหนักช่วงกลางวันวันธรรมดา

เครื่องมือนี้พัฒนาต่อยอดจาก `solar_payback_thailand` โดยนำเครื่องยนต์จำลองโหลดเครื่องใช้ไฟฟ้า, ตรรกะช่วงเวลา TOU, ดีไซน์ และกราฟมาใช้ซ้ำ และตัดส่วนที่เกี่ยวกับโซลาร์ทั้งหมดออก

---

## 🌟 คุณสมบัติหลัก (Key Features)

### 1. โหมดกรอกข้อมูล 2 แบบ (Detailed / Quick)
* **โหมดละเอียด:** กรอกเครื่องใช้ไฟฟ้าและเวลาเปิดใช้งานในตารางวันธรรมดา/วันหยุด ระบบจะคำนวณสัดส่วน Off-Peak ให้อัตโนมัติจากเวลาที่อุปกรณ์ทำงาน (ผ่านฟังก์ชัน `build24HourLoadProfile()` ที่ช่วงเวลา `p1`=กลางคืน → Off-Peak, `p2`+`p3`=กลางวัน-ค่ำ → Peak)
* **โหมดเร็ว:** กรอกแค่หน่วยไฟรวมต่อเดือนและประมาณสัดส่วนการใช้ไฟกลางคืนด้วยแถบเลื่อน (%)

### 2. อัตราค่าไฟแบบขั้นบันไดของมิเตอร์ปกติ (Progressive / Step Tariff)
* ค่าพลังงานของมิเตอร์ปกติคิดแบบ **ขั้นบันไดตามหน่วยรวมต่อเดือน** (0–150 / 151–400 / 401+ หน่วย) ไม่สนใจช่วงเวลา — ต่างจากมิเตอร์ TOU ที่คิดตามช่วงเวลา ค่าฐานต่อหน่วยทุกขั้นแก้ไขได้

### 3. เปรียบเทียบสองมิเตอร์และหาจุดคืนทุน
* **ค่าไฟปกติ/ปี** = `12 × (ค่าพลังงานขั้นบันได + Ft×หน่วย/เดือน + ค่าบริการ) × 1.07`
* **ค่าไฟ TOU/ปี** = `(หน่วย Peak×เรต Peak + หน่วย Off-Peak×เรต Off-Peak + Ft×หน่วยรวม) × 1.07 + ค่าบริการ×12×1.07`
* **ประหยัดต่อปี** = ค่าไฟปกติ − ค่าไฟ TOU (ติดลบได้ถ้า TOU แพงกว่า)
* **ระยะคืนทุน** = ค่าธรรมเนียมเปลี่ยนมิเตอร์ ÷ เงินที่ประหยัดได้ต่อปี (ถ้าประหยัดไม่ได้ → "ไม่คุ้ม")
* ค่าไฟทั้งหมดคิด **VAT 7%** เพิ่มจาก (ค่าฐาน + Ft) เหมือนบิลจริง MEA/PEA

### 4. กราฟและคำแนะนำ
* **กราฟโหลดรายวัน 24 ชม.** แยกสีช่วง Peak (ทอง) และ Off-Peak (เขียว) ให้เห็นชัดว่าการใช้ไฟอยู่ช่วงไหน
* **กราฟค่าใช้จ่ายสะสม** เส้นมิเตอร์ปกติ vs TOU (เส้น TOU เริ่มสูงกว่าเพราะรวมค่าเปลี่ยนมิเตอร์) จุดตัดคือจุดคืนทุน
* **คำแนะนำแบบภาษาคน** บอกว่าควรเปลี่ยนหรือไม่ พร้อมเหตุผล (เช่น สัดส่วน Off-Peak ต่ำเกินไป)

## 💻 การติดตั้งและเปิดใช้งาน (Setup & Usage)

1. เป็นแอปแบบ Single Page Application เขียนด้วย HTML, CSS และ Vanilla JavaScript ไฟล์เดียว
2. ดับเบิลคลิกเปิด `index.html` บนเบราว์เซอร์ยุคใหม่ได้ทันที ไม่ต้อง Build หรือติดตั้งเซิร์ฟเวอร์
3. กราฟใช้ไลบรารี **Chart.js** ผ่าน CDN

> **หมายเหตุ:** ค่าตั้งต้นของอัตราค่าไฟ ค่าบริการ และค่าธรรมเนียมเปลี่ยนมิเตอร์ เป็นค่าประมาณอ้างอิงและ **แก้ไขได้ทุกช่อง** โปรดตรวจสอบกับอัตราปัจจุบันของ กฟน./กฟภ. ก่อนตัดสินใจจริง
